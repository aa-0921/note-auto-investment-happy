**Automated Investment Content Sharing**

This application uses X API to automatically post investment-related content:

1. **Automated Posting**: Scheduled posting of Amazon investment product rankings and investment information with images (via GitHub Actions)

2. **Content Sharing**: Share investment strategies, market insights, and financial product information with the investment community

3. **Account Management**: Verify credentials and monitor rate limits

All content is for educational and informational purposes only. We comply with X API terms and rate limits.
